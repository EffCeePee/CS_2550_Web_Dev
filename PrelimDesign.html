Project Design

This page is dedicated to laying out the preliminary design of the project. This is not intended to be a final design but an initial idea on how the game will be laid out and played.

Constants: (in JavaScript these are just unchanging variables)

BEGINNER_SIZE: The size of the grid for a beginner game.
INTERMEDIATE_SIZE: The size of the grid for an intermediate game.
ADVANCED_SIZE: The size of an advanced game.
BEGINNER_BOMB_COUNT: The maximum number of bombs for a beginner game.
INTERMEDIATE_BOMB_COUNT: The maximum number of bombs for a intermediate game.
ADVANCED_BOMB_COUNT: The maximum number of bombs for a advanced game.

Objects:

Square
A square will have a _bomb, _adjacentBombs and a _flaged property.
  • _bomb: boolean property to determine if it holds a or not.
  • _adjacentBombs: if there is not a bomb it will hold the number of adjacent bombs
  • _flagged: tells if the sqare has been marked by a flag

Square object example:
{"_bomb": true, "_adjacentBombs" : null, "flagged": true}
{"_bomb": false, "_adjacentBombs" : 2, "flagged": false}
{"_bomb": false, "_adjacentBombs" : 0, "flagged": false}

Game
A Game will have a _bombCount property.
  • _bombCount : This will count the number of bombs flagged.

Game object example:
{"_bombCount" : 20}

Difficulty The Difficulty object will have _beginner, _intermdiate_, and _advanced properties.
  • _beginner: a boolean value that will be used when calling the difficulty function when the game is being set up.
  • _intermdiate_: a boolean value that will be used when calling the difficulty function when the game is being set up.
  • _advanced: a boolean value that will be used when calling the difficulty function when the game is being set up.

Clock

Functions:

Random(_size): this function will be to set up the status of each square in the game. It will take a size variable and randomly set the squares that will get bombs

ClockCount(): this function will be a counter that tells how much time it has taken to complete the game.

BombCount(_bombCount): this function will show how many bombs have not been flagged.

CountAdjacent(): This function will see how many bombs are in the adjacent squares.

Notes:

  1. This is a preliminary design and does not list all of the variables that will be used.
  2. There will also be other objects added or removed at a later time
  3. There will be more functions added later
